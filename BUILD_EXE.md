# Build para EXE (Windows) - Infinit War

## Requisitos
- Node.js 20+
- npm

## Comandos
1. `npm install`
2. `npm run dev` (teste desktop)
3. `npm run build:win`

O instalador `.exe` sera gerado em `dist/`.

## Observacoes
- Para publicar, adicione icone de app em `build/icon.ico` e configure no `package.json`.
- Para compras reais da loja, sera necessario backend + validacao de transacao.
